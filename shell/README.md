# shell

Shell configs and X session files.

| File | Purpose |
|---|---|
| `.zshrc` | Zsh: history, completions, pywal integration, plugins, aliases |
| `.bashrc` | Bash: ls/grep colors, PATH, nvm |
| `.bash_profile` | Sources .bashrc, PATH |
| `.xprofile` | Touchpad settings (natural scrolling, tapping, adaptive accel) — waits up to 5s for device |
| `.Xresources` | Includes `~/.cache/wal/colors.Xresources` |
| `.fehbg` | Wallpaper setter script |
